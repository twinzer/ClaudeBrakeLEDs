Powershell (windows default terminal)
    SSH into Pi:
    ssh pi@brakeled.local
    password is in pword mgr

    Navigate to your project and run the code:
    cd ~/ClaudeBrakeLEDs
    sudo python3 brake_leds.py
    
    restart LED service on Pi:
    sudo systemctl restart brakeleds.service

    Check the log 
    cat ~/ClaudeBrakeLEDs/brakeleds.log

    Test if Pi is running:
    ping brakeled.local
    
    sudo shutdown now
